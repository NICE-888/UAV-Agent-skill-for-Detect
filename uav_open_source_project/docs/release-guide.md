# GitHub 开源发布建议

## 推荐做法

1. 源码推送到 GitHub 仓库
2. 使用 MIT License
3. 使用 `.gitignore` 忽略 `*.rknn`
4. 在 GitHub Releases 中上传 `UAV.rknn`

## 建议的第一个版本

- Tag: `v1.0.0`
- Title: `RK3588 UAV Detector v1.0.0`

## Release 描述模板

```markdown
Initial open-source release.

Included:
- RK3588 UAV detection source code
- RKNN multithread inference pool
- Example scripts and documentation

Assets:
- UAV.rknn model weight
- Demo images
```
